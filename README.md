### comando: "dot -Tpng AFD.dot -o AFD.png" & "dot -Tpng AFN.dot -o AFN.png" 
### comando para los AFDs y AFNs: "dot -Tpng AFD<nombre del Token>.dot -o AFD<nombre del Token>.png" & "dot -Tpng AFN<nombre del Token>.dot -o AFN<nombre del Token>.png" 
El usar los siguientes comandos, luego de haber corrido el programa, generaran una imagen png del AFD y AFN y debe ejecutarse en el misma carpeta donde tiene generado el .dot. Recordar que se debe tener instalado Graphviz para que funcionen estos comandos
