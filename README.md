# Proyecto---Compiladores
_link proyecto 1: https://youtu.be/jQsVhQtmdjs
_comando: "dot -Tpng AFD.dot -o AFD.png" & "dot -Tpng AFN.dot -o AFN.png" 
_El usar los siguientes comandos, luego de haber corrido el programa, generaran una imagen png del AFD y AFN y debe ejecutarse en el misma carpeta donde tiene generado el .dot. _Recordar que se debe tener instalado Graphviz para que funcionen estos comandos
